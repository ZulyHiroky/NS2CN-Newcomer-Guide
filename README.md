--------------Q&A----------------反正也没人看，自问自答    
1.Q:加入这个项目我能干什么？  
  A:我也不知道你能干什么，可以把你会的东西都扔进去（?），也可以写一些你的个人心得。（比如smurf gorge）  
  
2.Q:我该怎么把想要的东西加进去？  
 A:可以单独做一个web把你的内容填进去，稍后会有专人（？）在主页上添加你的入口。  
 
3.Q:内容有什么限制吗？  
  A:请不要在内容中藏一些奇奇♂怪怪的东西。
