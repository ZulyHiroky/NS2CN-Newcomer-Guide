队伍的资源　　　　选中所有的士兵　　　　已经编号的单位　　　　　　　　　　　　　　　　　　　　　退出指挥官模式



这里是指挥官的操作界面，指挥官不直接参与战斗，采用类似即时战略游戏的控制方式来指挥战斗。
基本的操作同星际、魔兽之类，鼠标左键为选择、确定、开启开关，鼠标右键为对士兵下命令。
按Ctrl + 1-5 可以对选择的单位编队(含建筑)，编队后按 1-5 则选择被编队的单位。
右下方的12个指令的快捷键为QWERASDFZXCV这12个键(对应按键的位置)。

点右上方的 LOGOUT 可以离开指挥中心。
