极具战略性的NS模式　NS MODE
　　占矿、发展、拆基地，即时战略+第一人称射击
简单的解释就是以CS的视角去参与一场SC的战争。
NS模式的地图比较大，在该模式中玩家可以按C或M键查看全体地图
该模式中拥有资源、建筑的要素，双方都需要抢占资源点来进行发展并击败对方
胜利条件：彻底消灭敌对方的玩家（需要拆除复活相关的建筑使其无法复活）
对于PK爱好者的提醒：你不需要刻意地去寻找敌人来杀，这样做通常死的是你，你需要和队友配合并抢占有利的战略位置。这样等你的队伍把敌人封锁住以后（关厕所）就可以尽情地SM了。
