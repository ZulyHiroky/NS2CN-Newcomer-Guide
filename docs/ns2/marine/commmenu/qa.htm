指挥中心　Command Console
　 　 指挥中心　Command Console
造价: 15 res
造时: 15 sec
耐久: 3000 hp 1500ap e-hp：6000
前提建筑: 指挥站
用途: 允许队伍拥有指挥官
　　简称CC，允许一位Marine玩家进入充当指挥官的角色，当所有指挥站被摧毁，陆战队就输了，所以雷达是必需的。 只能放置在科技节点上，复活点IP必须建在它的周围。
- 基本功能 -
　
　　IP复活点必须建在他周围，指挥站是一切的开始。游戏开始的时候都有会一个指挥中心。当一位士兵进入指挥中心时(默认“Ｅ”键)，他便成为了指挥官，指挥官不直接参与战斗，采用类似即时战略游戏的辅助方式来指挥战斗、升级科技、分发装备、操控攻城坦克。如果没有指挥官，士兵就不再有新的建筑蓝图投放，无法获得新的装备，也就是说，他们无法发展！
　　同一时间内队伍中只能拥有一位指挥官。
- 专用能力 -
指挥站可以花费20资源解锁三合一（紧急供电、兴奋剂、纳米护盾）

　　回收该建筑，完全回收后获得该建筑造价70%的资源。
