开局及后续变化　How to start
此部分仅供参考，照搬的后果自负。



- 开局 -
开局的选择一般有3种。

　
Shotgun rush
　　广为人知的快攻战术。
　　开局的时候IP+Armory，并让士兵分散探出敌人的Hive所在，然后发Shotgun，冲Hive——胜利/失败。
　　……下面没有了，胜利就胜利了，失败就很难翻身了。
　　建议5v5或6v6的时候用。


搬家
　　在某些地图经常被使用的战术。
　　开局的时候什么也不造，让士兵尽快赶到新基地的位置，然后造CC，接着就如往常一般，通常要造TF来保护新基地。
　　……下面没有了，成功的话可以躲过敌人的快攻，并在战略上，失败的话基本没办法翻身。
　　搬家的目的地一般是：
　　RR（一个房间里有两个RT的）：可以很好地保护两个RT，从资源上获取优势。
　　Hive（当然是没建好的）：让敌人难以达成3Hive，从科技上获取优势。
　　交通要道：阻挡敌人的道路，一般是让他们走远路，从时间上获取优势。
　　注意，当你用OB的专用能力“戒严警报”的时候，士兵是被传回原来基地的位置的！


一般开局
　　正常的开局。
　　就是留在基地发展......
　　IP+Armory+OB+Arms Lab，十分标准的开局。



- 前中期过渡 -
为了中后期的发展，你需要做的有......

　
占矿
　　尽可能地占更多的矿。
　　占了矿当然要发展，这么多钱干啥用呢？
　　升级科技，一般1级的护甲是必升的，这样狗便需要多咬一口才可以咬死士兵。
　　不过某个版本开始因为初始护甲的提高，很多指挥官都改先升1级攻击了，1级护甲则用来防止被Focus Fade秒杀。
　　在交通要道建立阵地，关于这点请参照RT的相关战略。


骚扰
　　这里不是叫你去敌人基地骚扰。而是在占矿的同时去阻止敌人扩张。
　　尽可能拆除敌人的RT和阻止它们占更多的资源点。
　　遇到OC防守的地方请让士兵回避，在士兵多OC少的情况下（大约是5v3）可以用医药包辅助士兵强拆OC。


锁Hive
　　在敌人没还建Hive的地方建立阵地，请参照RT和PG的相关战略。
　　中途遇到OC请让士兵回避，OC少的话（1至2个）则用医药包辅助士兵强行通过，注意是通过不是强拆OC！


如何应对敌人的DC开局
　　DC开局，敌人的进化将会令它们拥有更强的自我回复能力、更高的护甲、而且“归巢”能力将使它们更难被消灭。
　　因为是前中期，敌人的一般不会有高级兵种出来，所以小兵们还是很容易对付的，重点在于DC本身的治疗能力。
　　一般Alien的矿会采取OC+DC的防守方式，这样的防守方式一般是无法强拆的，所以一般都用手雷或者是中华加农炮伺候！
　　至于一般异形，最好避免在DC的治疗范围内和它们交火，除非是用Shotgun秒。


如何应对敌人的MC开局
　　MC开局，敌人的进化将会令它们拥有更快的移动速度、更多的能量来进攻甚至无法听到它们发出的声音。
　　对于移动速度，可以用地雷来对付之，或者是机枪塔这种拥有自动瞄准能力的武器。
　　对于能量的问题，一般是强化蝙蝠的，然后就是毒气漫天......比较头疼，如没有OC保护的话就尽快消灭它，不然就回避吧。
　　对于无声，请避免单兵行动，集体行动的时候请注意互相照应，不要全部人都看一个方向。


如何应对敌人的SC开局
　　SC开局，敌人的进化将会令它们拥有隐形、热能感应甚至一击致命的能力。
　　对付的隐形方法当然是听声音啦。
　　热能感应......相当于被寄生了，所以你也别想去camp别人了。
　　敌人升集中的话，只要稍微辅助一下就可以秒杀了，那一级护甲升不升就自己决定吧。。
　　最后就是SC本身的能力——自带热能感应、自身及周围隐形、周边范围内OB的MT（动态扫描）无效化，这个主场优势真是NB。请让士兵集体行动，注意观察吧，OB的scan能力也不能少。



- 中期 -
只提点一下......

　
机枪塔
　　机枪塔威力很小，没必要就别造，浪费钱。


电磁防御
　　升级耗时又耗钱，威力一般，没必要就别升。


占矿&护矿
　　尽可能地占更多的矿，但更重要的是守住已经占领的矿。


占领交通要道
　　尽可能地占领更多的交通要道，这将为你赢得更多宝贵的时间。
　　在交通要道建立阵地，关于这点请参照TF和PG的相关战略。


攻防升级
　　升1级防，然后是3级攻。或者是1攻1防后满攻。


锁Hive
　　不要让敌人的第2、3Hive造起来。


霰弹枪
　　杀大虫必备。


传送门
　　叫那些偷偷潜进Hive的士兵不要打草惊蛇，然后造传送门......


高级武器
　　高级弹药库尽早升吧，3分钟的升级时间等你想升已经来不及了。


特殊防具
　　JP适合单兵作战，骚扰（进Hive骚扰），秘密潜入。
　　HA适合集体作战，集体推进，记得发Welder。


二择
　　升级攻防还是研究特殊防具？
　　人多的时候升级攻防提高整体战斗力，人少的时候研究特殊防具（一般是JP）。
