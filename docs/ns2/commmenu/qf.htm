指挥中心　Command Console
　 　 指挥中心　Command Console
造价: 20 res
造时: 15 sec
耐久: 10000 hp
前提建筑: 无
用途: 允许队伍拥有指挥官
　　简称CC，允许一位Marine玩家进入充当指挥官的角色，IP必须建在它周围。游戏开始时就有一个指挥中心了



- 基本功能 -
　
　　IP必须建在他周围，CC是一切的开始。游戏开始的时候都有会一个指挥中心。当一位士兵进入指挥中心时(默认“Ｅ”键)，他便成为了指挥官，指挥官不直接参与战斗，采用类似即时战略游戏的控制方式来指挥战斗、升级科技、分发装备。如果没有指挥官，士兵们就不能在新建任何建筑，无法获得新的装备，也就是说，他们无法发展！
　　同一时间内队伍中只能拥有一位指挥官。



- 专用能力 -
　
　　回收该建筑，完全回收后获得该建筑造价1/2的资源。
