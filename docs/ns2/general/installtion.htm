安装　Installtion
　　安装游戏，基本上就是下载安装包――双击运行――选择安装路径（建议不要安装在C盘）――下一步......



系统要求　System Requirements
  1. 要有 "Steam" 或者是用 "WON"（CS1.5用的那个）。现在WON已经停止服务了，而且NS对WON只支持到2.0，所以Steam才是王道啊。
    • Steam: 你必须先安装Steam。想了解更多关于Steam的东西点这里，含下载的哦。
    • WON: 你必须先安装 Half-life、Counter-strike Retail、或者 Day of Defeat Retail 其中一个或多个，并且将Half-life的引擎升级到1.1.1.0（CS升到1.5就对了）。
  2. 你至少要有一台 P2-400 级的 PC、Windows 95的操作系统、32M 的内存、一张图形加速卡（现在谁还没有这东西-_-b，板载显卡都没问题，建议在OpenGL 下用 1024x768 来玩）。最后至少要有 300 M 的硬盘空间。
  3. 你得上网，至少 56kbps 的网速（用猫的话认命吧）。
  4. 不是必须，但最好是有: 了解一下什么叫 "第一人称射击" （比如 《Quake》 或 《CS》） 和 "即时战略" （比如 《星际争霸》 或 《Command & Conquer》）。这些东西对你进行游戏很有帮助。



安装Steam　Steam Installtion
　　点击这里进入Steam的下载页面
　　默认是装在C盘，但是因为HL的引擎文件也会装在Steam的安装目录下（700M左右），所以建议不要装在C盘



安装游戏　Game Installtion
　　点击这里进入NS客户端的下载页面
　　默认是装在 Steam的安装目录\SteamApps\你的帐号\half-life\ 下



视频设置　Vedio Option
　　建议使用OpenGL方式，因为其它两种方式下的准心显示有错误
